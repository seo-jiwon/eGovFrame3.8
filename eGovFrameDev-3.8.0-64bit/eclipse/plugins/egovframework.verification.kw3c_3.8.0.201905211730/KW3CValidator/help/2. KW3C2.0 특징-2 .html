KW3C 2.0 특징

새로 제작된 웹표준 진단 프로그램은 프로젝트 관리 기능이 대폭 개선되었습니다. 아울러 이전 버전에서 불가능하였던 HTML5 평가 기능도 추가되었습니다.

 

주요 개선 사항은 다음과 같습니다.

  1. 프로젝트의 생성, 재편집이 가능합니다. 따라서 평가를 다시 수행하기 위하여 새로 프로젝트를 생성할 필요가 없이 기존의 프로젝트를 수정하여 평가할 수 있습니다.
  2. 동일한 이름으로 프로젝트를 생성하거나 재편집 또한 평가를 수행하는 경우에 기존의 평가결과는 그대로 보존됩니다. 이때 기존 프로젝트와 평가결과는 번호가 자동으로 붙어 프로젝트 관리가 용이합니다.
  3. 프로젝트마다 적용되는 기본 설정을 하고, 프로젝트 생성 시에 이 값이 적용됩니다. 또한 프로젝트 별로 설정 값을 변경할 수 있는 기능을 제공함으로써 프로그램의 사용성을 대폭 강화하였습니다.
  4. KW3C2.0은 장애인 접근성을 제공합니다. 모든 기능은 메뉴로 선택이 가능합니다. 또한 각 창에서도 Tab키, Space키 및 Enter키를 이용한 키보드 접근성을 100% 제공합니다.
  5. 평가 프로세스를 실기한 확인할 수 있도록 프로그래스 바와 로그 창을 추가하였습니다. 프로그램의 오동작시에 로그 창의 기록을 담당자에게 통보하면 빠르고 편리한 대응을 받을 수 있습니다.
  6. 웹 형식의 오프라인 도움말을 제공합니다. 도움말은 이 프로그램을 사용하는데 필요한 사용법과 각종 정보를 제공합니다.
  7. W3C의 DTD 평가 페이지와 CSS 평가 페이지를 별도의 화면으로 제공하여 W3C 웹 페이지를 통한 평가를 손쉽게 할 수 있습니다.
