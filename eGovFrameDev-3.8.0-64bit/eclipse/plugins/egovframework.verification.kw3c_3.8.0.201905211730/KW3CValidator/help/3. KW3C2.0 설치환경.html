KW3C 2.0 설치환경

 

기본 요구 사항

  • 하드웨어
    • CPU: 펜티엄 IV 2 GHz 이상
    • 메모리: 1,024MB 이상
    • 하드디스크: 200MB 이상(평가할 웹사이트가 늘어남에 따라 하드디스크 용량이 늘어나야 함)
    • 네트워크: KW3C2.0의 모든 기능을 사용하기 위해서는 PC가 네트워크에 연결되어야 합니다.  
  • 운영체제
    • Windows XP Service Pack 3
    • Windows Server 2003 Service Pack 2
    • Windows Server 2008 Service Pack 2
    • Windows Vista Service Pack 2
    • Windows7  

    추가 설치 프로그램  

    KW3C2.0을 안정적으로 실행하기 위해서는 다음의 프로그램이 반드시 설치되어 있어야 합니다.

    • JAVA : 스타일 시트 평가를 위해서는 JAVA 최신 버전이 설치되어야 됩니다. 단, 설치버전은 32bit 버전이어야 합니다. 64비트 버전을 설치한 경우에는 정상적으로 동작하지 않습니다.
    • Microsoft .NET Framework 4 : KW3C2.0은 .NET기반으로 개발되었습니다. 따라서 KW3C2.0을 사용할 PC에 .NET 4.0을 설치하지 않으면 KW3C2.0이 실행되지 않습니다.

    ※ 이 프로그램은 Internet Explorer 7 이상의 버전에서 사용이 가능합니다. 그 이하의 버전에서는 정상적으로 동작하지 않습니다. 또한 Windows XP 환경에서는 사용하는 웹브라우저에 따라 필요한 보안 패치를 다운받아 설치해야 합니다. Windows 7에서는 보안패치가 불필요합니다.

    • IE 7용 보안패치(KB2699988) 바로가기
    • IE 8용 보안패치(KB2699988) 바로가기
