기존 프로젝트 평가 결과 보기  

 

이전에 평가한 프로젝트를 열어 점검결과를 다시 확인할 수 있습니다.  

1) 평가 결과 보기를 위해서는 다음 중 한 가지 방법이 가능합니다.  

  • 메뉴의 [파일]-[프로젝트보기]를 선택
  • 시작페이지 왼쪽의 [최근점검한 프로젝트]에서 프로젝트 명을 선택
  • 툴바에서 [열기]아이콘 선택  

 

프로젝트 결과 보기를 실행하면 [프로젝트 열기] 창이 열립니다.

 

   

 

2) 프로젝트 이름을 선택하고 더블 클릭을 하거나 [열기] 버튼을 누르면 평가결과 창이 열립니다. 평가결과 창의 이용 방법은 평가 결과 보기에서 설명한 바와 같습니다.  

 

기존 프로젝트 삭제

 

이전에 평가한 프로젝트를 삭제 할 수 있습니다.  

1) 평가 결과 보기를 위해서는 다음 중 한 가지 방법이 가능합니다.  

  • 메뉴의 [파일]-[프로젝트삭제]를 선택
  • 툴바에서 [삭제]아이콘 선택  

 

프로젝트 삭제를 실행하면 [프로젝트 삭제] 창이 열립니다.

 

 

 

2) 프로젝트 이름을 선택하고 더블 클릭을 하거나 [삭제] 버튼을 누르면 삭제확인 창이 뜨고 [예] 버튼을 누를 시 해당 프로젝트가 삭제됩니다.

[모두삭제] 버튼을 누르면 삭제확인 창이 뜨고 [예] 버튼을 누를 시 표시되어있는 모든 프로젝트가 삭제됩니다.
