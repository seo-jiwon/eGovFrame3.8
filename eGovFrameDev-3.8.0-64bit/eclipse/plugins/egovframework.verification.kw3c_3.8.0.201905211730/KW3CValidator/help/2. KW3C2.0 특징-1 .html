KW3C 2.0 특징

W3C 웹표준 문법진단

  • W3C의 웹표준 문법검사 기능을 제공하고 검사결과를 점수로 환산하여 제시합니다. 오류에 따른 점수 환산 방법은 7. 전자정부서비스 웹표준 진단표를 참고하십시오

 

W3C 스타일 시트 문법진단

  • W3C의 CSS 문법검사 기능을 제공하고 검사결과를 점수로 환산하여 제시합니다. 오류에 따른 점수 환산 방법은 7. 전자정부서비스 웹표준 진단표를 참고하십시오.

 

원격시스템 및 로컬시스템

  • 웹사이트를 통해 제공되는 웹페이지 및 PC에 저장되어 있는 웹 소스 파일에 대해서 웹표준 문법과 CSS 문법의 오류를 측정할 수 있습니다.

 

프로젝트 관리 기능

  • KW3C2.0은 하나의 웹사이트 또는 파일 묶음을 프로젝트 단위로 관리하여, 언제든지 프로젝트의 작성 및 수행, 재편집, 삭제 그리고 평가결과 보기 등이 가능합니다.
  • 동일한 이름의 프로젝트를 반복적으로 수행하는 경우에는 먼저 평가한 프로젝트 정보와 평가결과가 백업파일로 저장되므로 주기적인 프로젝트의 수행 및 관리가 가능합니다.

 

평가결과 보기 기능

  • 웹표준 문법 및 스타일 시트 문법 평가 결과는 오류 정보와 그 내용을 웹 페이지 소스와 함께 보여주며, 상호 연동이 가능하므로 오류 수정의 평의성을 제공합니다.
