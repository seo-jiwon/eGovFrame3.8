접근성 기능

 

KW3C2.0은 장애인이 사용할 수 있도록 키보드 장애인 접근성을 제공합니다. 키보드 접근성은 다음과 같습니다.

1) 주메뉴

아이콘으로 선택 가능한 기능은 모두 메뉴를 통해 제공이 가능합니다. 또한 자주 사용하는 개별 메뉴 기능은 단축키로도 접근이 가능합니다. 주메뉴 및 단축키는 다음과 같습니다.  

  • 파일 : Alt+F
  • 프로젝트생성 : ALT+F, N
  • 프로젝트편집 : ALT+F, E
  • 프로젝트보기 : ALT+F, O
  • 프로젝트삭제 : ALT+F, D
  • 종료: ALT+F, X 또는 ALT+F4  
  • 설정 : Alt+O
  • 웹소스 수집 방법 : ALT+O, O
  • 평가항목설정 : ALT+O, H
  • 웹페이지 확장자 설정 : ALT+O, T  
  • 창 : Alt+W
  • 창: ALT+W, W  
  • 도움말 : ALT+H
  • KW3C 도움말 : ALT+H, H 또는 F1
  • 관련홈페이지가기 : ALT+H, G
  • KW3C 정보 : ALT+H, --> A

 

2) 결과 보기 탐색

결과 보기 화면에서 사용할 수 있는 단축키는 다음과 같습니다.  

  • 탭의 이동

순서결과보기 화면에서 Tab키의 이동순서는 다음과 같습니다. Tabl키를 누를 때마다 URL목록 -> 소스페이지 탭 -> 열린 소스페이지 탭의 소스창 -> 오류영역 -> 전체화면 탭-> URL목록 의 순서로 이동합니다. Shift+Tab 키에 의한 순서는 Tab키와 반대로 이동합니다. 여기서 전체화면 탭은 KW3C2.0 사용 시 열린 창을 선택하는 페이지 탭입니다.  

  • 영역의 탐색
    • URL목록 영역

URL목록영역이 선택되어 있으면 키보드의 상하 이동키를 이용하여 URL 및 CSS항목을 선택할 수 있습니다. 해당 항목의 소스와 오류내역을 확인하려면 Space키를 누릅니다. 해당 항목의 하위항목을 열거나 닫으려면 Enter키를 누릅니다.  

  • 소스보기 영역

소스보기 영역은 상하좌우키로 탐색이 가능합니다. 또한 일부영역을 선택하여 복사도 가능합니다.

  • 탐색 : 키보드 상하좌우키
  • 영역선택 : 영역의 시작지점에서 Shift키를 누르고 상하좌우키를 눌러 이동시킵니다.
  • 영역복사 : Ctrl+C키를 누르면 선택된 영역이 복사되어 ‘노트패드’등에 붙혀넣기가 가능 합니다.  
  • 오류영역

오류영역의 도표는 상하좌우키를 이용하여 탐색이 가능합니다. 이때 Space키를 누르고 있는 동안 해당소스 페이지 탭이 선택되고 해당 줄과 열의 위치가 하이라이트 됩니다. Space키를 놓으면 원상태로 돌아옵니다.
