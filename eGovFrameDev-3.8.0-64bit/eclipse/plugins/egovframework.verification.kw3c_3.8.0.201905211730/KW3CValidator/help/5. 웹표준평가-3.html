웹 표준 평가

프로젝트 실행(웹표준 검사)  

프로젝트 ‘검사 설정’ 창에서 ‘시작’ 버튼(프로젝트 생성/열기 그림의 [버튼⑤])을 누르면 웹표준 평가가 시작되어 다음과 같은 프로젝트 실행 창이 나타납니다. 프로젝트 창의 프로그레스바는 프로젝트의 진행 정도를 보여줍니다. 중앙의 그리브 뷰에는 웹페이지주소 또는 파일별로 작업 현황을 보여줍니다.  

  • - 표시 : 작업이 진행되지 않았음을 알려줍니다.
  • √ 표시 : 작업이 완료되었음을 알려줍니다.  

 

 

[자세히] 버튼을 누르면 자동점검의 상세한 진행 상황을 보여주는 창이 화면에 열리고 [자세히] 버튼을 다시 누르면 사라집니다.  

 

 

평가가 완료되면 다음과 같은 평가 완료 화면으로 변경되고 [확인] 버튼이 나타납니다. [확인] 버튼을 누르면 평가결과 보기 화면으로 이동합니다.
